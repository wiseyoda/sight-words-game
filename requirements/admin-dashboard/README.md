## Key Features

### 1. Player Management

- Create up to 5 player profiles
- Switch between players
- Individual settings per player
- Delete/reset player data

### 2. Progress Reports

- Word mastery visualization
- Time played tracking
- Mission completion stats
- Struggling word alerts

### 3. Content Library

- View all words (built-in + custom)
- View all sentences
- Add custom words
- Upload custom audio

### 4. AI Content Generator

- Generate new sentences from topics
- Generate full campaigns
- Generate audio for new words
- Custom theme creation

### 5. Settings

- Audio volume controls
- Gameplay adjustments
- Data export/import
- Account management

## Mobile Considerations

Dashboard is parent-focused but should work on:
- Tablets (primary)
- Desktop (secondary)
- Phones (limited, but functional)

Responsive layouts prioritize:
1. Readability
2. Touch targets
3. Core functionality
