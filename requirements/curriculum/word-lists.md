## Word Selection Algorithm

### For Mission Generation

When generating sentences, select words based on:

1. **Priority 1**: Words child is "learning" (needs practice)
2. **Priority 2**: New words (introduce gradually)
3. **Priority 3**: Mastered words (maintain, but less frequent)

### Word Mix Per Mission

| Word Type | Percentage |
|-----------|------------|
| Learning (needs practice) | 40% |
| New (introducing) | 30% |
| Mastered (review) | 30% |

### Distractor Selection

Distractors should:
- Be from same difficulty level
- Be same part of speech when possible
- Not create valid sentence if used
- Be visually distinct (not confusing)
