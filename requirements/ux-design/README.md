## Quick Reference

### Touch Targets

| Element | Minimum | Preferred |
|---------|---------|-----------|
| Word cards | 64x64px | 80x60px |
| Buttons | 48x48px | 64x48px |
| Navigation | 44x44px | 56x56px |

### Typography

| Element | Size | Weight |
|---------|------|--------|
| Word cards | 28-36px | Bold |
| Headings | 24-32px | Bold |
| Body text | 18-24px | Regular |
| Labels | 14-16px | Medium |

### Colors

| Purpose | Contrast Ratio |
|---------|----------------|
| Text on background | 4.5:1 minimum |
| Large text | 3:1 minimum |
| Interactive elements | 3:1 minimum |

---

## Core Principles

1. **Large Touch Targets** - Easy for small fingers
2. **No Reading Required** - Icons + audio for navigation
3. **Immediate Feedback** - Response < 100ms
4. **Forgiving Input** - Generous hit areas, easy undo
5. **Safe Environment** - No external links, no ads
